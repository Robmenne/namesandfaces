---
title: "Date Group Switch"
description: "Date Range Switches can be used to allow your users to group charts by selected time intervals"
---

A Date Group Switch can be linked to charts to configure which time interval the chart is grouped by. This allows your users to select daily, weekly, monthly, or yearly views of their data.

The date group switch can only be linked to certain charts such as bar charts and line charts. It also requires a the data to be grouped by a date field.

Example of configured Date Group Switch

To configure a date group switch, first drag the component onto the dashboard.

Next, select the chart you want to link, and click on the arrow beside the date field to group on. Select Date Group Swith under user inputs.

Select the correct component from the dropdown below.

Your chart will now be configured to change based on the data group selected.
