Dashboards are defined as a collection of components. These include data panels, text, date selectors, dropdown menus, and more.

* Drag and drop these components onto your dashboard to use them.

* To edit a component, **double-click on the component** and the editor will appear in the left side panel.

## Adding a new component

Drag a component onto the canvas to add it to your dashboard.

Adding new components is easy! Simply drag the component onto the canvas to get started. You will see a hovering shadow, indicating where the component will go when you drop it.
